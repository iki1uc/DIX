# DIX – Data Index Modul

DIX ist das Datenindexierungsmodul des iki1uc-Systems.
Es strukturiert Daten für IN, IO, OUT und SCAN-HUB.

## Aufgaben
- Daten indexieren
- Daten sortieren
- Daten für IO bereitstellen
- Daten für SCAN-HUB vorbereiten
- Zeit-Raster interpretieren

## Achse
Data-Achse (D-Achse)
